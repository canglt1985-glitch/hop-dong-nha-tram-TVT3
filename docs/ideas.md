# Dự án: Soạn thảo Phụ lục Hợp đồng BTS

## Ý tưởng ban đầu
- Tự động hóa việc tạo các file Phụ lục hợp đồng (.docx) từ dữ liệu danh sách hợp đồng (.xlsx).
- Sử dụng các file mẫu: `DNCM02.docx`, `DNCM11.docx`, `DNCM24.docx`.
- Nguồn dữ liệu: 
    - Danh sách hợp đồng (Excel).
    - Bảng đơn giá/hạng mục: `DS1245 PLHD.xlsx`.
    - Dữ liệu địa chỉ mới: `TOAN BO DU LIEU_1278_TRAM_cang.letan_20260421_091229.xlsx`.

## Mục tiêu
- Giảm thiểu sai sót thủ công khi nhập liệu.
- Rút ngắn thời gian soạn thảo văn bản.
- Cập nhật địa chỉ xã/phường mới chính xác theo dữ liệu chuẩn.
