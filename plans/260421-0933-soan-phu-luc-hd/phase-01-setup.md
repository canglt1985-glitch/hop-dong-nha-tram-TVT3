# Phase 01: Project Setup

## Objective
Thiết lập môi trường phát triển cho cả Backend (Python) và Frontend (React).

## Tasks:
- [ ] Khởi tạo thư mục dự án chuẩn.
- [ ] Setup Backend: Tạo môi trường ảo venv, cài đặt các thư viện (fastapi, pandas, openpyxl, docxtpl, uvicorn).
- [ ] Setup Frontend: Khởi tạo project Vite (React + TypeScript), Tailwind CSS.
- [ ] Cấu hình API Proxy để Frontend gọi được Backend.
- [ ] Kiểm tra kết nối cơ bản (Hello World).

## Output:
- Cấu trúc thư mục sạch sẽ.
- React chạy được ở port 5173, FastAPI chạy ở port 8000.
